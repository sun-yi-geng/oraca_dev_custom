# oraca_dev_custom

## How to use
You can find "release" on the right side. Select the latest or the desired version, and then download and install it to use it.

你可以在右侧找到release，下载您需要的版本，或者下载最新版本，安装后即可使用，生成的配置文件夹基本在 \%appdata%\OrcaSlicer\ 


## How to develop
Pull the official OrcaSlicer to the local machine, and then copy the Goofoo folder and Goofoo.json located in the OrcaSlicer\resources\profiles path in the installed OrcaSlicer to the local repository.(Under the same path)

拉取官方的orcaslicer到本地，再将安装好的orcaslicer中的OrcaSlicer\resources\profiles路径下的Goofoo文件夹和Goofoo.json复制到本地仓库（相同路径下）

Refer to this official document:https://www.orcaslicer.com/wiki/developer_reference/how_to_build#windows-64-bit

参考此官方文档：https://www.orcaslicer.com/wiki/developer_reference/how_to_build#windows-64-bit
